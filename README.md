## Quick Start

```bash
# with pnpm
pnpm add rsrv
pnpm exec rsrv -p 3000 public assets/css # Starts the server on port 3000, serves ./public and ./assets/css

# with npm
npm install -g rsrv
npx rsrv -p 3000 public assets/css
```
